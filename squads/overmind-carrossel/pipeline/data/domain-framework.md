# Domain Framework: Carrosséis Instagram B2B de IA

> Squad: overmind-carrossel | Marca: Overmind

---

## Framework Operacional

### Fase 1: Pesquisa e Curadoria
1. Buscar notícias, tendências e dados recentes sobre IA aplicada a negócios
2. Filtrar por relevância para o público-alvo (CEOs, COOs, gestores de PMEs)
3. Priorizar por potencial de engajamento: dado surpreendente > opinião contrária > tutorial prático
4. Rankear top 3-5 histórias com título, fonte, data e resumo

### Fase 2: Seleção e Angulação
1. Usuário seleciona UMA notícia/tema
2. Gerar 5 ângulos distintos sobre a mesma história usando drivers psicológicos diferentes
3. Cada ângulo usa um tipo diferente de hook (dado, pergunta, erro, contrário, antes/depois)
4. Usuário seleciona o ângulo

### Fase 3: Criação do Carrossel
1. Diagnóstico pré-escrita (awareness level, sophistication, Big Idea, driver psicológico)
2. Escolher formato de carrossel (Editorial, Listicle, Tutorial, Mito vs Realidade, etc.)
3. Escrever 3 opções de hook para o slide 1
4. Após seleção do hook, escrever copy de todos os slides (40-80 palavras/slide)
5. Escrever caption (hook nos primeiros 125 chars + body + CTA + hashtags)
6. Criar design system visual (cores Overmind, tipografia, grid)
7. Gerar HTML de cada slide (1080x1440, self-contained)
8. Renderizar e verificar visualmente

### Fase 4: Revisão
1. Avaliar copy contra critérios de qualidade
2. Avaliar design contra critérios visuais
3. Verificar brand voice Overmind
4. Emitir APPROVE ou REJECT com feedback acionável

### Fase 5: Publicação
1. Preview completo (imagens + caption + hashtags)
2. Dry-run (validar credenciais e formato)
3. Confirmação do usuário
4. Publicar e reportar URL do post

---

## Critérios de Decisão

### Quando usar qual formato de carrossel
- **Editorial/Tese:** Tendências, análise de mercado, dados surpreendentes
- **Listicle:** Ferramentas, dicas, recursos, checklists
- **Tutorial:** Como fazer X com IA, passo a passo
- **Mito vs Realidade:** Desconstruir crenças erradas sobre IA
- **Antes/Depois:** Cases, transformações, resultados
- **Storytelling:** Histórias pessoais, jornada de clientes
- **Problema/Solução:** Dores do público → como IA resolve

### Quando escalar ao usuário
- Notícia controversa que pode afetar a imagem da marca
- Dado sem fonte verificável
- Tema fora do escopo de IA/negócios
- Revisor rejeitou 3 vezes o mesmo conteúdo
