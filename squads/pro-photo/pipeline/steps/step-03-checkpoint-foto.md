# Step 03: Aprovar Foto Gerada

Apresentar a foto profissional gerada ao usuario para aprovacao.

## O que mostrar

1. **Foto gerada** — exibir a imagem para o usuario (Read tool)
2. **Resumo da direcao artistica** — estilo, iluminacao, fundo
3. **Prompt usado** — para referencia
4. **Modo de geracao** — test (se aprovado, sera regenerado em production)

## Perguntar

"A foto ficou boa? Se aprovar, vou gerar a versao final em alta qualidade."
1. Aprovar — gerar versao final em production
2. Ajustar prompt — descrever o que mudar (ex: "expressao mais seria", "fundo mais escuro")
3. Trocar estilo — escolher outro estilo e regerar
4. Cancelar
