# Domain Framework: Video → Business Plan Pipeline

## Fluxo Operacional Completo

### Fase 1: Extração de Conteúdo
1. Receber URL do YouTube e contexto do usuário (briefing)
2. Acessar o vídeo e extrair transcrição/conteúdo completo
3. Organizar o conteúdo em roteiro estruturado com seções temáticas
4. Identificar os pontos-chave: problema apresentado, solução proposta, público mencionado, modelo de negócio (se houver)
5. Se vídeo de software: capturar screenshots das telas mostradas

### Fase 2: Formulação da Ideia
1. Preencher Lean Canvas a partir do conteúdo extraído
2. Mapear Value Proposition Canvas (Jobs/Dores/Ganhos)
3. Formular a ideia como hipótese falsificável
4. Identificar lacunas de informação que a pesquisa de mercado precisa preencher

### Fase 3: Pesquisa de Mercado
1. Dimensionar TAM via relatórios de indústria (Statista, IBISWorld, etc.)
2. Narrowar para SAM com critérios geográficos e segmento
3. Estimar SOM realista para primeiro ano
4. Mapear concorrentes diretos, indiretos e substitutos
5. Analisar Porter's Five Forces
6. Compilar SWOT com evidências concretas
7. Responder "por que agora?" — timing do mercado

### Fase 4: Plano de Negócio
1. Redigir Sumário Executivo (standalone, 1 página)
2. Detalhar modelo de negócio e fontes de receita
3. Calcular unit economics (CAC, LTV, ARPU)
4. Definir go-to-market: primeiros 100 clientes, primeiro canal
5. Projetar financeiro com premissas explícitas
6. Documentar riscos e mitigações

### Fase 5: Plano de Execução
1. Definir MVP via MoSCoW
2. Criar roadmap faseado (3 horizons)
3. Estimar recursos: time, budget, tech stack
4. Definir métricas de sucesso por fase
5. Estabelecer kill criteria — quando pivotar ou parar

### Fase 6: Revisão
1. Verificar consistência entre todos os documentos
2. Validar que dados de mercado suportam as conclusões do plano
3. Conferir que MVP é genuinamente mínimo
4. Verificar que riscos estão documentados honestamente
5. Scoring por critérios de qualidade definidos

## Critérios de Decisão

| Sinal | Classificação | Resposta |
|-------|--------------|----------|
| TAM > $1B e crescendo > 15%/ano | Mercado atrativo | Recomendar exploração agressiva |
| SAM < $10M | Mercado nicho | Avaliar se nicho permite rentabilidade |
| > 5 concorrentes diretos bem financiados | Alta competição | Exige diferencial claro ou pivô |
| Nenhum concorrente direto encontrado | Validar demanda | Pode não haver mercado — investigar |
| Modelo de receita não evidente no vídeo | Lacuna crítica | Pesquisar modelos similares do setor |
