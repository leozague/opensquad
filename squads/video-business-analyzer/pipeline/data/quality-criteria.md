# Quality Criteria: Video Business Analyzer

## Critérios por Documento

### Roteiro do Vídeo
- [ ] Conteúdo completo — nenhum trecho significativo omitido
- [ ] Organizado em seções temáticas com timestamps
- [ ] Pontos-chave destacados separadamente do corpo
- [ ] Citações diretas preservadas entre aspas

### Ideia de Negócio
- [ ] Expressa como hipótese falsificável ("Segmento X pagará Y por Z")
- [ ] Lean Canvas completo (9 blocos preenchidos)
- [ ] Problema validado como real, não assumido
- [ ] Modelo de receita concreto (não "monetizar depois")
- [ ] Cliente-alvo específico (cargo, demográfico, comportamento)
- [ ] Lacunas de informação listadas explicitamente

### Análise de Mercado
- [ ] TAM sourced (Statista, IBISWorld, relatórios setoriais)
- [ ] SAM narrowed com critérios claros
- [ ] SOM realista para primeiro ano
- [ ] Mínimo 3 concorrentes diretos e 2 indiretos mapeados
- [ ] SWOT com itens específicos e baseados em evidências
- [ ] "Por que agora?" respondido com dados
- [ ] Todas as fontes citadas com URL e data de acesso
- [ ] Nível de confiança atribuído a cada finding

### Plano de Negócio
- [ ] Sumário executivo standalone (legível isoladamente)
- [ ] Unit economics calculados (CAC, LTV, ARPU)
- [ ] Projeções financeiras com premissas explícitas
- [ ] Go-to-market específico (primeiros 100 clientes)
- [ ] Riscos documentados honestamente
- [ ] Modelo de receita detalhado com pricing

### Plano de Execução
- [ ] MVP genuinamente mínimo (resolve 1 problema core)
- [ ] Roadmap faseado com horizons definidos
- [ ] Cada fase com métrica de sucesso mensurável
- [ ] Recursos estimados (time, budget, timeline)
- [ ] Tech stack justificado por velocidade e capacidade
- [ ] Kill criteria definidos

## Scoring

| Score | Significado |
|-------|-------------|
| 9-10 | Excelente — pronto para usar |
| 7-8 | Bom — melhorias opcionais |
| 5-6 | Aceitável — necessita ajustes |
| < 5 | Rejeitar — refazer |

**Threshold de aprovação:** Score médio >= 7/10, nenhum critério individual abaixo de 4/10.
